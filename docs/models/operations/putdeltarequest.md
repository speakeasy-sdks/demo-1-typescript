                                 | An array of Deltas.<br/><br/>The Deltas in the request are combined, meaning the current Delta is updated in turn by each Delta in the request. Once all Deltas have been combined, the resulting Delta is simplified.<br/><br/>* All Modules in the `modules.add` property are replaced with the new Delta's values. If the value of a Module is `null`, and the ID is in the `modules.remove` list, it is removed from the `modules.remove` list.<br/><br/>* All IDs listed in `modules.remove` are combined. Any ID in `modules.remove` and also in `modules.add` are removed from `modules.add`<br/><br/>* The lists of JSON Patches in `modules.update` are concatenated or created in `modules.updates`.<br/><br/>Simplification involves:<br/><br/>* Applying any entries in `modules.updates` that have matching IDs in `modules.add` to the `modules.add` entry and removing the `modules.update` entry.<br/><br/>* Reducing the number of JSON Patches in each `modules.update` entry to the smallest set that has the same effect.<br/><br/>**Extension to JSON Patch**<br/><br/>If a JSON Patch entry needs to be removed, without side effects, the `value` of the `remove` action can be set to `{"scope": "delta"}. This will result in the remove action being used during simplification but be discarded before the Delta is finalized.<br/><br/>If the user making the request is not the user who created the Delta and they are not already on the contributors list, they will be added to the contributors list.<br/><br/>_NOTE: If the `id` or `metadata` properties are specified, they will be ignored._ |
